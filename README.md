# ⚡ Farhan Malik — Bold Frontend Developer Portfolio

Welcome to a portfolio that doesn't believe in boring whitespace or soft pastel colors. 
This is a high-contrast, loud, neo-brutalist playground built to showcase frontend engineering with absolute intent.

### 🛠️ What's under the hood?
* 🚀 **Angular 21 + TypeScript Signals** for high-velocity state management.
* 🦄 **Tailwind CSS v4** for utility-first styling with attitude.
* 📦 **Nx Monorepo & Jest** for robust, bulletproof architecture.
* 🎨 **Zero Images** for branding—just CSS, text-based interactive elements, and custom typography.
